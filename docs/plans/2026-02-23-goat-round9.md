**Step 2: Add GOAT spec docstrings to all 13 public methods**

Format required:
```python
def method_name(self, df: pl.DataFrame, ...) -> ReturnType:
    """One-line summary.

    Input:
        df: DataFrame to operate on; returns None/empty if None
        param2: description and constraints

    Output:
        description of return value and structure

    Raises:
        QueryError: specific condition that triggers it
        pl.exceptions.InvalidOperationError: if column dtype incompatible

    Invariants:
        - result row count <= input row count (for filter)
        - None return iff df is None
    """
```

Key contracts per method:
- `filter`: raises `QueryError` if operator unknown; returns `None` if df is None; NaN rows excluded from GT/GE/LT/LE
- `sort`: returns `None` if df is None; may raise `pl.exceptions.InvalidOperationError` for bad column names
- `group_aggregate`: returns `None` if df is None; result columns named `{col}_{func}`
- `get_statistics`: returns `{}` silently if column not in df; result dict has `n`, `mean`, `std`, `min`, `max`, `median`, etc.
- `get_all_statistics`: returns `{}` if df is None; only numeric columns by default
- `get_full_profile_summary`: returns `None` if not windowed and profile is None
- `is_column_categorical`: returns `False` if df is None or column not found
- `get_unique_values`: returns `[]` if df is None or column not found
- `sample`: returns `None` if df is None
- `get_slice`: returns `None` if df is None
- `search`: returns empty DataFrame if no matches; searches ALL columns
- `filter_by_map`: returns `None` if df is None; missing keys silently skipped
- `create_index`: returns empty dict if df is None or column not found
