## Task 1: Narrow `except Exception` blocks in `ipc_server.py`

**Why:** Auditor flagged 4 over-broad catches. Three are cleanup handlers that only touch file/network I/O — should catch `OSError`. One handles JSON + write errors — should catch `(json.JSONDecodeError, OSError)`.

**Files:**
- Modify: `data_graph_studio/core/ipc_server.py:111,158,164,170`
- Modify tests (if any exist): `tests/unit/test_ipc_*.py`

### What to change

**Line 111 (stop → port file cleanup):**
```python
# Before:
except Exception:
    logger.debug("ipc_server.stop.port_file_unlink_failed", exc_info=True)

# After:
except OSError:
    logger.debug("ipc_server.stop.port_file_unlink_failed", exc_info=True)
```

**Line 158 (_handle_client outer catch — JSON decode + write errors):**
```python
# Before:
except Exception as e:
    logger.warning("ipc_server.handle_client_error", extra={"error": str(e)})

# After:
except (json.JSONDecodeError, OSError) as e:
    logger.warning("ipc_server.handle_client_error", extra={"error": str(e)})
```

**Lines 164 and 170 (writer error write / writer close):**
```python
# Before:
except Exception:
    logger.debug("ipc_server.handle_client.error_write_failed", exc_info=True)
# ...
except Exception:
    logger.debug("ipc_server.handle_client.writer_close_failed", exc_info=True)

# After:
except OSError:
    logger.debug("ipc_server.handle_client.error_write_failed", exc_info=True)
# ...
except OSError:
    logger.debug("ipc_server.handle_client.writer_close_failed", exc_info=True)
```
