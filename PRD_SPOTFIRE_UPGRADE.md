## 2. 🔗 마킹 & 필터링 시스템 (Spotfire-style)

### 2.1 Marking (마킹) 개념

Spotfire의 마킹은 단순 선택을 넘어서는 강력한 데이터 연동 메커니즘입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│  📍 MARKING SYSTEM                                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Marking: 여러 시각화에서 동일한 데이터 선택을 공유하는 메커니즘       │
│                                                                      │
│  ┌─────────────────┐      ┌─────────────────┐                       │
│  │   Scatter Plot  │      │    Bar Chart    │                       │
│  │    ●  ●        │      │    ▓▓▓         │                       │
│  │   ●● ●●       │      │    ▓▓▓ ▓▓▓     │                       │
│  │  ●●●●●●       │──────│    ▓▓▓ ▓▓▓ ▓▓▓ │  ← 같은 마킹 공유     │
│  │ ●●● [선택] ●●  │      │    ▓▓▓ [선택]   │                       │
│  │●●●●●●●●●●    │      │    ▓▓▓ ▓▓▓ ▓▓▓ │                       │

│  ┌─────────────────────────────────────────────────────────────┐    │

│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.2 다중 마킹 지원

```python
# 마킹 설정 구조
@dataclass
class Marking:
    name: str                    # 마킹 이름 (예: "Main", "Comparison")
    color: str                   # 마킹 색상
    selected_rows: Set[int]      # 선택된 행 인덱스

@dataclass
class MarkingSettings:
    markings: Dict[str, Marking]  # 여러 마킹 정의
    active_marking: str           # 현재 활성 마킹
```

**다중 마킹 사용 사례:**
- **Main Marking**: 주 분석 대상 선택
- **Comparison Marking**: 비교 대상 선택
- 두 마킹 간 교집합/차집합 분석

### 2.4 Data Limiting (데이터 제한)

시각화별로 표시할 데이터를 마킹 또는 필터링으로 제한:

```python
class DataLimiting(Enum):
    ALL_DATA = "all"                    # 전체 데이터
    FILTERED = "filtered"               # 현재 필터 적용
    MARKED = "marked"                   # 마킹된 데이터만
    FILTERED_AND_MARKED = "both"        # 필터 + 마킹

@dataclass
class VisualizationLimiting:
    filtering_scheme: str               # 사용할 필터링 스킴
    limit_by_marking: Optional[str]     # 마킹으로 제한
    show_empty_when_no_marking: bool    # 마킹 없을 때 비우기
```

### 3.2 Trellis 모드

```python
class TrellisMode(Enum):
    ROWS_AND_COLUMNS = "rows_columns"  # 행과 열로 분할
    PANELS = "panels"                   # 페이지 형태로 분할

@dataclass
class TrellisSettings:
    mode: TrellisMode
    row_axis: Optional[str]            # 행 분할 컬럼
    column_axis: Optional[str]         # 열 분할 컬럼
    panel_axis: Optional[str]          # 패널 분할 컬럼 (PANELS 모드)
    max_rows: int = 5                  # 최대 행 수
    max_columns: int = 5               # 최대 열 수
    show_empty_panels: bool = True     # 빈 패널 표시 여부
    manual_layout: bool = False        # 수동 레이아웃
```

## 4. 📈 Lines & Curves (트렌드 라인 및 회귀분석)

### 4.1 지원할 커브 피팅 모델

Spotfire 수준의 회귀분석 기능:

| 모델 | 수식 | 용도 |
|------|------|------|
| **Straight Line** | y = ax + b | 선형 관계 |
| **Polynomial** | y = Σ(aᵢxⁱ) | 비선형 곡선 (차수 선택) |
| **Exponential** | y = a·eᵇˣ | 성장/감쇠 |
| **Power** | y = a·xᵇ | 거듭제곱 관계 |
| **Logarithmic** | y = a·ln(x) + b | 로그 관계 |
| **Logistic (4PL)** | y = d + (a-d)/(1+(x/c)ᵇ) | 용량-반응 곡선 |
| **Gaussian** | y = a·exp(-((x-b)²/2c²)) | 정규 분포 |
| **Moving Average** | - | 이동 평균 |
| **LOWESS** | - | 비모수 평활화 |
| **Holt-Winters** | - | 시계열 예측 |

├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Function Name: [Outlier Detection                  ]               │
│                                                                      │
│  Script:                                                            │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  import numpy as np                                          │    │
│  │  from scipy import stats                                     │    │
│  │                                                              │    │
│  │  def detect_outliers(data, column, threshold=3):            │    │
│  │      z_scores = stats.zscore(data[column])                  │    │
│  │      outliers = np.abs(z_scores) > threshold                │    │
│  │      return outliers                                         │    │
│  │                                                              │    │
│  │  result = detect_outliers(input_data, 'Sales')              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  Input Parameters:                                                  │
│  ─────────────────────────────────────────────────────────────      │
│  input_data: [Current Selection ▼]  Type: DataFrame                │
│  threshold:  [3.0              ▼]  Type: Float                     │
│                                                                      │
│  Output:                                                            │
│  ─────────────────────────────────────────────────────────────      │
│  ☑ Add as calculated column: [Is_Outlier]                          │
│  ☐ Replace marking                                                  │
│                                                                      │
│  [Test Run]  [Save Function]  [Execute]                             │
│                                                                      │

@dataclass
class Marking:
    """Spotfire 스타일 마킹"""

    """다중 마킹 관리"""

    def mark(self, marking_name: str, indices: Set[int], mode: MarkMode) -> None
    def get_marked(self, marking_name: str) -> Set[int]
    def clear_marking(self, marking_name: str) -> None

    # 시그널
    marking_changed = Signal(str, set)  # marking_name, indices

@dataclass
class FilteringScheme:
    """필터링 스킴"""
    name: str
    filters: List[FilterCondition]

class FilterManager:
    """다중 필터링 스킴 관리"""
    schemes: Dict[str, FilteringScheme]

    def apply_filter(self, scheme_name: str, condition: FilterCondition) -> None
    def get_filtered_data(self, scheme_name: str) -> pl.DataFrame

    # 시그널
    filter_changed = Signal(str)  # scheme_name

class CurveFitter:
    """커브 피팅 엔진"""

    @staticmethod
    def fit_linear(x: np.ndarray, y: np.ndarray) -> CurveFitResult

    @staticmethod
    def fit_polynomial(x: np.ndarray, y: np.ndarray, degree: int) -> CurveFitResult

    @staticmethod
    def fit_exponential(x: np.ndarray, y: np.ndarray) -> CurveFitResult

    @staticmethod
    def fit_logistic(x: np.ndarray, y: np.ndarray) -> CurveFitResult

    ) -> Tuple[np.ndarray, np.ndarray]
